Join the Start Bootstrap Community on Slack!

Start Bootstrap now has a public workspace on Slack. You can join the community at https://startbootstrap-slack.herokuapp.com/. This is the best place to get help and to communicate with Start Bootstrap!

Accessing the Themes & Templates

To use the themes and templates available on Start Bootstrap, you can download them directly from the website using the download button on each of the template overview pages. You can also find the source files for each template or theme on GitHub.

Still Having Trouble?

If you run into any bugs with any of the templates or themes, please feel free to open an issue on GitHub in the respective items repository. There is a comments section on each page that you can look through or post to if you have a question.

The best way to get help with an issue though is to join the Start Bootstrap community on Slack and ask your question there. You can join the community at https://startbootstrap-slack.herokuapp.com/.
